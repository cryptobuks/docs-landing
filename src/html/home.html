    Data Explorer

    Data explorer reads from the primary unless the primary is unavailable. If the primary is unavailable, Data Explorer reads from a non-hidden, non-delayed secondary member.
  • Import Tools

    MongoDB Atlas Live Import

    MongoDB Atlas can perform live migration of a source replica set to an Atlas cluster.

    mongomirror

    mongomirror is a utility for migrating data from an existing MongoDB replica set to a MongoDB Atlas replica set.

    mongoimport / mongorestore

    The mongorestore program writes data from a binary database dump created by mongodump to a MongoDB instance.
